# Europejska Akademia Lotnictwa — Web Platform

This repository houses the source code for the digital business card of **Europejska Akademia Lotnictwa** (`eal.world`). 

## Technical Stack
- **Frontend:** Semantic HTML5, Tailwind CSS (via CDN)
- **Hosting:** GitHub Pages
- **DNS:** Managed via custom apex domain configuration

## Architecture & Security Notes
- **Static Delivery:** The site contains zero server-side execution, minimizing the attack surface to zero.
- **Dependency Management:** Built using utility-first CSS via a locked CDN endpoint to ensure lightning-fast loading speeds and zero build-time vulnerability risks.
